# Security Policy

This repository is a governance template.

If you discover a security issue in the template itself, use the repository's
private security reporting feature or contact the maintainers listed by the
hosting platform. Do not disclose vulnerability details in a public issue.

For adopters:

- This file is intentionally a template boundary, not a claim that this
  repository has an adopter-specific response SLA or security contact.
- Before production adoption, replace this file with the adopting
  repository's private vulnerability reporting path, response expectations,
  supported versions, and disclosure policy.
- Keep release, install, and provenance evidence aligned with your published policy.
