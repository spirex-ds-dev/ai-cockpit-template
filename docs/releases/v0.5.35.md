# v0.5.35

v0.5.35 is the current template release. It includes the completed Bootstrap Adoption, Project Configuration, Governed Development, lifecycle-fact, cross-stack validation, delusion-regression, and multilingual documentation alignment work.

## Release boundary

The published release workflow binds the immutable tag, source commit, release assets, SBOM, provenance, digest manifest, workflow run, and strict smoke verification. Repository-local candidate metadata is not provider evidence. This release does not claim trusted identity, sandbox isolation, immutable audit, enterprise security, compliance certification, or production readiness.

## Adoption note

Adopter projects must create their own installation or upgrade Work Item from their own remote default branch and consume the published `v0.5.35` tag. Installation and upgrade remain separate adopter-project PRs with their own review and lifecycle closure.
