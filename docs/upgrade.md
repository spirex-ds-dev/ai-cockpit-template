# Upgrade

This compatibility entry now lives at [Reference Upgrade](reference/upgrade.md).

Updates require an Impact Assessment before recalibration. An update is not Ready merely because Runtime installation succeeds. The existing Active Configuration remains in force until the Candidate configuration activates successfully; failed activation must leave the old configuration available.
